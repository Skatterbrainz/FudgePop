# Install-Chocolatey

## SYNOPSIS
Insure Chocolatey is installed

## SYNTAX

```
Install-Chocolatey [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Check if Chocolatey is installed. 
If not, then install it.

## EXAMPLES

### EXAMPLE 1
```
Install-Chocolatey
```
